# Technomart

Учебный проект интернет-магазина электроники, выполненный в рамках курса [HTML Academy](https://htmlacademy.ru/).

## Демо

https://f1eeman.github.io/technomart/

## Технологии

- HTML5
- SASS/SCSS
- Gulp 4

## Запуск

```bash
npm install
npm start
```

Сайт откроется на `http://localhost:3000` с автоперезагрузкой при изменениях.

## Сборка

```bash
npm run build
```
